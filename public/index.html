Sign in with GitHub

Your mind maps will be saved as private JSON files in a repository called mindspark-maps on your own GitHub account. This app talks to GitHub directly — there's no server in between.

  1. Create a personal access token on GitHub →
    Pick classic token, scope: repo (so the app can create the private repo and read/write files in it).
How does this work / is it safe?

MindSpark runs entirely in your browser. The token is stored only in localStorage on this device, and is sent only to api.github.com — never to any third party.

On first sign-in the app creates a private repo called mindspark-maps in your account. Every save commits a small JSON file to that repo, so your data is versioned and you can git clone it any time.

To revoke access: github.com/settings/tokens
